# Materi Kal
Web ini berisi catatan saya selama mempelajari mata kuliah Komputasi Aljabar Linear (KAL).

Komputasi Aljabar Linear sendiri adalah cabang ilmu yang mempelajari bagaimana konsep-konsep aljabar linear dihitung dan diselesaikan menggunakan metode numerik serta bantuan komputer.

Secara umum, yang dipelajari meliputi antara lain:

1. Sistem Persamaan Linear

  => Mempelajari cara menyelesaikan sistem persamaan berbentuk:

    Ax=b

  => Metode yang dipelajari:

    - Eliminasi Gauss

    - Gauss-Jordan

    - Faktorisasi LU

    - Metode Iteratif (Jacobi, Gauss-Seidel)

 => Digunakan dalam:

    - Perhitungan teknik

    - Analisis rangkaian listrik

    - Pemodelan ekonomi

2. Matriks dan Operasinya

    - Penjumlahan dan perkalian matriks

    - Invers matriks

    - Determinan

    - Transpose

    - Aplikasi:

    - Transformasi geometri

    - Grafik komputer

    - Machine Learning

3. Vektor dan Ruang Vektor

    - Operasi vektor

    - Basis dan dimensi

    - Kombinasi linear

    - Ruang kolom dan ruang baris

4. Nilai Eigen dan Vektor Eigen

    - Digunakan untuk:

    - Analisis stabilitas sistem

    - Google PageRank

    - Machine Learning (PCA)

5. Dekomposisi Matriks

 => Teknik penting dalam komputasi:

    - LU Decomposition

    - QR Decomposition

    - SVD (Singular Value Decomposition)

    - SVD banyak dipakai dalam:

    - Kompresi gambar

    - Pengolahan sinyal

    - Data Science

6. Norma dan Aproksimasi

    - Norma vektor dan matriks

    - Least Squares (Metode Kuadrat Terkecil)

    - Error dan stabilitas numerik

7. Implementasi dengan Komputer

    - Biasanya menggunakan:

    - Python (NumPy, SciPy)

    - MATLAB

    - Octave
